## Description

### Lab 01：简易计算器I

- 主要内容：
  - 理解 stack 数据结构
  - 实现字符栈功能
  - 实现表达式中数值与操作符的识别
  - 实现表达式中括号匹配的判断
- 实验输入：带有大中小括号的四则混合运算表达式
- 实验输出：依次输出每个操作符的名称，以及操作数的值。输出过程中如果出现括号匹配错误则报错然后退出输出。
